# HandDetection

Курсовой проект по обнаружению рук в видеопотоке с веб-камеры с помощью
собственной нейронной сети.

Приложение получает изображение с камеры в реальном времени, определяет
положение рук и отображает найденные области с помощью ограничивающих рамок.

## Возможности

- получение видеопотока с веб-камеры;
- обнаружение рук в реальном времени;
- отображение сетки предсказаний модели;
- построение ограничивающих рамок вокруг найденных рук;
- фильтрация пересекающихся рамок с помощью Non-Maximum Suppression;

## Модель

В основе детектора используется `EfficientNetV2B0`.

Модель работает по grid-based принципу: изображение размером `224 × 224`
разделяется на сетку `7 × 7`. Для каждой ячейки модель предсказывает:

- смещение центра объекта по осям X и Y;
- ширину и высоту ограничивающей рамки;
- принадлежность к фону или классу `hand`;
- уверенность в наличии объекта.

После восстановления координат рамок применяется Non-Maximum Suppression,
который удаляет повторяющиеся пересекающиеся предсказания.

Модель обучалась в течение 200 эпох.

## Запуск

Клонируйте репозиторий:

```bash
git clone https://github.com/Whiox/HandDetection.git
cd HandDetection
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Скачайте готовую модель и поместите файл `hand_detection.keras`
в корневую папку проекта.

Запустите приложение:

```bash
python main.py
```

Для завершения работы нажмите клавишу `Q` в окне видеопотока.
